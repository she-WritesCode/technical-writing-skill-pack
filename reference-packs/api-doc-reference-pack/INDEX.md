# API Doc Reference Pack

This pack is a set of reusable references for drafting API documentation with AI.

Use these files together:

1. Start with `source-inventory-template.md`
2. Use `intake-question-bank.md` to fill gaps
3. Pick a shape from `doc-templates.md`
4. Draft with `drafting-patterns.md`
5. Review with `verification-checklist.md`

Recommended rule: do not ask AI to write a final draft until the source inventory is at least mostly complete.
